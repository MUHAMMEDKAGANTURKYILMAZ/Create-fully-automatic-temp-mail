## PROJECT OVERVIEW
This is a lightweight and efficient **Python script** designed to automate the process of creating temporary email addresses. Unlike complex web structures, this tool runs directly in your terminal or as a background process to fetch emails quickly without any manual overhead ⚡.

---

## CORE FEATURES
* **Rapid Generation:** Creates unique temporary mail addresses in seconds through the command line 🚀.
* **Terminal Based:** No browser or heavy UI needed; everything is handled via Python logic 🐍.
* **Automated Cleanup:** Automatically handles the expiration and deletion of used mailboxes 🧹.
* **Direct Integration:** Can be imported into other Python projects as a standalone module 🔗.
